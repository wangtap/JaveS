final 关键字：final关键字表示最终，不可变

	  被final修饰的方法，不可以被复写

static 关键字；静态的
静态的属性和方法是类的
特也就是static修饰的属性，或者方法是类的 
 不是依赖于对象的
 
 1,静态方法中不能使用非静态的变量
 2静态的方法或者属性，可以理解为共享的状态
 每个对象的操作的都是一个相同的属性或者方法
 注意是相同的，不是两个一模一样的东西
 指的是在你内存中的，就存在一个
 3静态的属性和方法，可以使用类名直接调用
 eg：
 static  int a；
 static void show (){}
 类名.a
 类名，show
